  • 전체 단계 진행률 항상 표시

  • 마지막 단계에 Aha 배치

  • 계약서 없어도 시작 가능

UI 상태 & 엣지 케이스

각 화면의 로딩/오류/빈 상태와 예외 케이스 처리 설계

로딩 상태 (Loading States)

AI

AI 판정 중

📄

문서 생성 중

프로그레스 바. 단계별 ("분석 완료 → 초안 생성 중") 표시

📊

대시보드 로딩

스켈레톤 UI. 카드별 독립 로딩으로 부분 표시

빈 상태 (Empty States)

0

프로젝트 없음

"첫 프로젝트를 만들어보세요" + 샘플 데이터 체험 CTA

0

판정 기록 없음

"첫 요청을 판정해보세요" + 빠른 시작 가이드 링크

0

계약서 미등록

"계약서를 등록하면 더 정확한 판정이 가능합니다" + 등록 유도

오류 상태 (Error States)

!

AI 판정 실패

재시도 버튼 + "문제가 지속되면 고객지원" 링크. 자동 재시도 1회

!

계약서 파싱 실패

수동 입력 모드로 전환 유도. 파싱 불가 이유 간단 설명

!

네트워크 오류

오프라인 감지 배너. 재연결 시 자동 재시도
