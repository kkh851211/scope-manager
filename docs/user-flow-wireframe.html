# 화면명 페르소나 진입 조건 핵심 액션 검증 지표
S1 Landing Hero 👩‍💻실무자 외부 채널 유입 CTA 클릭 / 데모 체험 Bounce Rate < 60%
S2 회원가입 👩‍💻실무자 Hero CTA 클릭 소셜/이메일 가입 가입 완료율 > 60%
S3 프로필 & 가이드 👩‍💻실무자 가입 완료 직무 선택, 예시 확인 이슈 입력 진입율 > 70%
S4 이슈 입력 ⭐ 핵심 👩‍💻실무자 온보딩 완료 자유 텍스트 1~3개 작성 입력 완료율 > 50%
S5 AI 처리 중 👩‍💻실무자 "AI 분석 시작" 클릭 대기 (단계별 피드백 확인) 이탈율 < 10%
S6 AI 결과 & 검토 ⭐ 핵심 👩‍💻실무자 AI 처리 완료 금액 확인·수정·첨부·확정 PDF 진행율 > 40%
S7 이슈 목록 (보관함) 👩‍💻실무자 로그인 후 홈 이슈 추가·편집·피치덱 생성 WAU 15% → 30%
S8 피치덱 미리보기 ⭐ 핵심 👩‍💻실무자 "PDF 생성" 클릭 PDF 다운로드 / 링크 공유 Activation 달성 지점
S9 Pricing (B2B 탐색) 🏢경영진 콜드메일 / 피치덱 수령 데모 요청 폼 제출 데모 요청 월 10건
S10 B2B 대시보드 개발 예정 🏢경영진 WTP 검증 후 개발 ROI 트렌드 확인 / 알림 수신 Free-to-Paid CVR
